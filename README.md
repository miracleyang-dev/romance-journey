# Romance Journey

一个轻量的静态网页项目，用于展示与“浪漫旅程”主题相关的页面与交互逻辑，适合作为静态展示页或原型演示。

## 功能概述

- 主题化的页面展示与基础交互
- 前端逻辑与样式分离，便于维护

## 项目结构

- index.html
- css/style.css
- js/app.js
- js/lunar.js
- js/store.js

## 运行方式

- 直接用浏览器打开 index.html
- 或使用任意静态服务器工具进行本地预览

## 开发建议

- 修改样式请在 css/style.css
- 交互逻辑集中在 js/app.js
- 需要持久化或本地缓存的逻辑可放在 js/store.js
- 与日期或农历相关的逻辑在 js/lunar.js

## 部署方式

这是纯静态项目，可部署到任意静态托管平台（如 GitHub Pages、Netlify、Vercel）。

## 注意事项

- 浏览器缓存可能影响更新，必要时请强制刷新
